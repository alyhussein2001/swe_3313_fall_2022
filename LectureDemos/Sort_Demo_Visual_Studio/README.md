# swe_3313_fall_2022 Lecture Demos
## See ~/Lecture_Demos/Sort_Demo_Visual_Studio

```git clone https://github.com/jeff-adkisson/swe_3313_fall_2022.git```

Open in `sort-demo.sln` from Lecture Demos directory in Visual Studio 2022 and run.

# Detailed Steps:
1. Install git. https://github.com/git-guides/install-git
2. Install Visual Studio 2022 Community Edition and include .Net Desktop Development when selecting the libraries to install. https://visualstudio.microsoft.com/vs/community/
3. Open a terminal/command prompt.
4. Switch to a directory to hold the downloaded demo.
5. Type Type ```git clone https://github.com/jeff-adkisson/swe_3313_fall_2022.git```
6. You can open the .sln file in Visual Studio to view/run the code.
7. You can run the code from the command line by switching to the `Lecture Demos / sort-demo` directory (which contains `sort-demo.csproj`) and type `dotnet run`.

# Mac Users
If you are a Mac user, you need a Windows Developer virtual. I recommend using VirtualBox or Parallels on a Mac. Once you have the Windows virtual machine operational, follow the steps above to install Visual Studio 2022 and git.
